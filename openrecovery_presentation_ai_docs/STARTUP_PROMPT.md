# Startup Prompt For A New AI Session

Copy and paste this into a fresh AI session:

```text
Please help me continue creating OpenRecovery Reveal.js HTML5 lesson decks.

First read these docs:
- README.md
- PROJECT_CONTEXT.md
- PRESENTATION_CONSISTENCY_REQUIREMENTS.md
- STYLE_GUIDE.md
- OPENRECOVERY_IMAGE_CREATION_GUIDE.md
- SLIDE_TEMPLATES.md
- CURRENT_TASK.md
- DECISIONS.md
- SESSION_LOG.md
- QA_CHECKLIST.md

Then:
1. summarize the active task
2. tell me the next best step
3. continue the work
4. keep the deck style consistent with PRESENTATION_CONSISTENCY_REQUIREMENTS.md and shared-styles/master.css
5. update CURRENT_TASK.md, SESSION_LOG.md, and DECISIONS.md if needed

I may provide the lesson source as a local file path, a Google Doc link, or pasted content.
Infer the lesson metadata from that source unless I explicitly override it.
By default, include `3–5` images using OPENRECOVERY_IMAGE_CREATION_GUIDE.md unless I explicitly ask for fewer or none.
```
