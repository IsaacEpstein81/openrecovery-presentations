# End Of Session Prompt

Copy and paste this before ending an AI work session:

```text
Before we end this session:

1. Update CURRENT_TASK.md with the current status and next step.
2. Append a short note to SESSION_LOG.md.
3. Update DECISIONS.md if any durable design, repo, style, or implementation decision was made.
4. Update QA_CHECKLIST.md if we discovered a new recurring issue to check.
5. Tell me:
   - what changed
   - what is next
   - any blockers or open questions
   - any files I should commit
```
